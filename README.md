Examples of code of Aera Project (http://aetestingtools.com)
Author: Pavel Amialiushka

Aera is software for data analysis of acoustic emission testing,
that has been developed for nondestructive testing laboratory of OJSC “Naftan”. 

Aera uses C++98, boost, multitasking, WTL. 

The code was developed using Visual Studio 98, but it also can be compiled using Visual Studio 2015. 

Aera has x32 and x64 versions.
